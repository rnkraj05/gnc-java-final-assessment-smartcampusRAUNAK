# 🎓 Smart Campus Management System (Java)

## 📌 Project Overview

This project is developed as part of the final assessment for Java training.
It simulates a Smart Campus System where an admin can manage students, courses, and enrollments.

The main goal of this project is to apply core Java concepts in a real-world scenario.

---

## 🚀 Features Implemented

### 👨‍🎓 Student Management

* Add student (ID, Name, Email)
* View all students
* Uses OOP concepts (Class, Object, Constructor)

---

### 📚 Course Management

* Add course (Course ID, Course Name, Fee)
* Validates course fee using custom exception

---

### 🔗 Enrollment System

* A student can enroll in multiple courses
* Used **HashMap + ArrayList**
* Efficient data storage and retrieval

---

### 📋 Menu Driven Program

* User-friendly console menu
* Options:

  * Add Student
  * Add Course
  * Enroll Student
  * View Students
  * View Enrollments
  * Process Enrollment
  * Exit

---

### ⚠️ Exception Handling

* Handled invalid input using try-catch
* Created custom exception: `InvalidFeeException`
* Prevents program crash and ensures reliability

---

### ⚙️ Multithreading

* Simulated enrollment processing using Thread
* Used `Thread.sleep()` to create delay
* Shows background processing concept

---

### 💾 File Handling (Bonus)

* Data stored in file (`student.txt`)
* Used FileWriter to save data
* Used BufferedReader to read data

---

## 🧠 Concepts Used

* Classes & Objects
* Constructors
* OOP Principles
* Collections (HashMap, ArrayList)
* Exception Handling
* Multithreading
* File Handling

---

## ▶️ How to Run

1. Open project in VS Code
2. Compile:

   ```
   javac -d ..\bin *.java
   ```
3. Run:

   ```
   java -cp "..\bin" SmartCampus
   ```
